# 🧠 Polyphasic Biological Clock (PBC)

> **"Optimizing the Human Runtime Environment"**

## 📖 What is the Polyphasic Biological Clock?

Traditional sleep is a monolithic process—a long, uninterrupted block of downtime that can be inefficient for high-performance cognitive tasks. The **Polyphasic Biological Clock (PBC)** is an engineering approach to sleep. 

Instead of one long "system shutdown," PBC breaks sleep into several smaller, strategically timed "reboot cycles" (naps). This is modeled after how high-availability computing systems handle maintenance: 
instead of shutting down the entire server, we perform micro-updates and cache clears during low-traffic periods.

### The Core Philosophy:
- **Throughput Optimization:** Minimizing long periods of inactivity.
- **Context Switching:** Using scheduled naps to clear "neural cache" (adenosine buildup) and reset cognitive focus.
- **Predictability:** Eliminating the anxiety of "when to sleep" by providing a deterministic, time-locked schedule.

By using **Unix Timestamps** and **Precision Countdown**, this extension acts as your external "System Kernel," keeping your biological clock synchronized with your productive output.(for time sensetive activities :))

## 🛠 Manual Installation Guide

Since this is a developer-centric tool, it is distributed as a "Raw Source" package. 
Follow these steps to deploy the extension to your browser.

### 1. Prepare the Source Code
First, you need to get the files onto your local machine.
Download the project as a .rar file from GitHub and extract it to a folder on your computer.

2. Deploy to Browser (Brave/Chrome)
Because this is not an official store app, you must use the Developer Mode to “sideload” the extension.

Open Extensions Page:

In Brave: Type brave://extensions in the URL bar.
In Chrome: Type chrome://extensions in the URL bar.
Enable Developer Mode:

Locate the “Developer mode” toggle in the top-right corner and turn it ON.

Load the Unpacked Extension:

Click the “Load unpacked” button (usually in the top-left).

A file navigator will appear.
Select the folder where you extracted the files (the folder that contains the manifest.json file).

Click Open or Select Folder.

3. Activation
You should now see the Polyphasic Clock icon in your extension bar.

Tip: Click the “Puzzle” icon and Pin the extension to your toolbar for instant access to your countdown.

Select your protocol (Uberman, Everyman, etc.) and hit “Start Schedule”.
